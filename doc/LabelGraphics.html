LabelGraphics

new LabelGraphics(options)

Name Type Description
options LabelGraphics.ConstructorOptions 初始化对象参数
Example:
const entity = Map3d.Entity({
     position: Map3d.Position.toCartesian(new Map3d.Position(123, 23, 100)),
     label: {
         text: "hello,world",
     }
});
viewer.entities.add(entity);
