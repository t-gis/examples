Viewer

new Viewer(container, options)

Creates an instance of Viewer.
Name Type Description
container HTMLElement | string | ol.Map 容器
options Viewer.ConstructorOptions 初始化参数
Example:
// 1、兼容老的 sdk
const mapObj = new gis.Map2D();
mapObj.createMap(....);

const viewer = new Map2d.Viewer(mapObj.map);

// 2、直接使用新的 sdk
const viewer = new Map2d.Viewer("map");
