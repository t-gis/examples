Methods

static WebFeatureServiceImageryProvider.load(url, options) → GeoJsonDataSource

加载 wfs 图层
Name Type Description
url String 请求的 url
options Object 具有以下属性
Name Type Description
layerName String 图层名称
Returns:
Example:
// wfs 服务链接
const url = "http://139.9.191.233:8090/iserver/services/data-GD/wfs100/utf-8";
// 请求参数
const options = {
	// 图层名称
	layerName: "GD:树林林地"
}
// 加载链接，得到数据源
const dataSource = WFSLayer.load(url, options)
// 添加数据源
mapObj.addDataSource(dataSource)
