根据经纬度数组数据调整合适的视角
Name Type Default Description
positions Array.<Array.<Number>> 经纬度二维数组
padding Array.<Number> [10,10,10,10] optional 矩形视角内边距

// 定义经纬度数组数据

    [114.03302290725708, 22.613155727386474],
    [114.07297709274292, 22.646844272613524],
    [114.053, 22.63]
]

// 调整视角
map2d.camera.fit(positions, [50, 50, 50, 50])

flyTo(options)

设置相机位置
Name Type Description
options Object 对象具有以下属性
Name Type Description
destination Position 将要设置的位置信息
duration Number 动画的持续时间以毫秒为单位
complete function 动画结束回调函数
Example:
// 创建终点
const position = new Position(114.07291271972656, 22.646887187957763)

// 调整视角
map2d.camera.flyTo({
    destination: position,
    duration: 3000,
    complete: () => {
        console.log("complete")
    }
})
